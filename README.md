# ADK Slides Map — Arialexis

Interactive Folium/Leaflet map of 69 climbing slides across 20 Adirondack High Peaks.

Features:
- OSM polygons, CalTopo curated traces, Wikiloc track overlays (off-trail clipped)
- Per-slide info panel (photo, access, difficulty, GPS source, beta links)
- Mountain summit labels (▲) declutter at low zoom
- Approximate-position markers tagged with "?" badge

Generated by `build_all_slides_map.py` (see source repo). Public mirror only.
